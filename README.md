# DoS Detector System

Система обнаружения DoS-атак с использованием нейронных сетей.

## Структура проекта
- `dos_detector.py` - основной модуль детектора
- `neural_network_95_percent.py` - нейросетевая модель
- `model_trainer.py` - обучение моделей
- `packet_processor.py` - обработка сетевых пакетов
- `live_capture.py` - захват трафика в реальном времени
- `config.yaml` - конфигурационные параметры
